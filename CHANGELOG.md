# Changelog

## v0.2.0 — 2026-04-04

- **Ask AI** — New "Ask AI" button in Clean, Prune, and Optimize modules that opens ChatGPT with a prefilled prompt about your selected items, so you can get advice on what's safe to remove
- **Coach mark** — One-time hint bubble guides first-time users to discover the Ask AI feature
- **URL limit guard** — Ask AI button auto-disables with a tooltip when too many items are selected to fit in a browser URL

## v0.1.0 — 2026-04-04

Initial release.

- **Clean** — Scan and remove system caches, user caches, logs, browser data, developer tool artifacts, and orphaned app data
- **Prune** — Detect and bulk-delete developer artifacts (node_modules, target, __pycache__, .build, etc.) across your projects
- **Installers** — Find and remove leftover .dmg, .pkg, and .iso installer files
- **Uninstaller** — Fully remove apps along with their associated support files, caches, and preferences
- **Analyze** — Treemap disk visualizer, large file finder, and duplicate file detector
- **Status** — Real-time CPU, memory, disk, and network monitoring dashboard
- **Settings** — Launch at login, trash mode, dry run, ignore list, low disk alerts, auto-updates
- **Onboarding** — Guided setup with animated demo, FDA permission, and quick preferences
