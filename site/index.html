Questions you might have.

Is it really free?

Yes. Kyra is free and open source under the MIT license. No trials, no subscriptions, no "premium" tier.

Is it safe?

Kyra moves files to Trash by default, so nothing is permanently deleted. You can always recover. The code is fully open on GitHub.

What Macs does it support?

Any Mac with Apple Silicon (M1, M2, M3, M4) running macOS 11 Big Sur or later.

Why does macOS say it's damaged?

Kyra isn't notarized by Apple yet. Go to System Settings, Privacy & Security, and click "Open Anyway." We're working on getting it signed.
